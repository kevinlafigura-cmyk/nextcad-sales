# NextCAD Sales Website

Sitio web de ventas y marketing para NextCAD.

## Características

- Diseño Profesional Moderno con paleta azul
- Tema Claro optimizado para conversión
- Navegación Sticky
- Secciones: Hero, Problema/Solución, Características, Integración, Precios, CTA

## Stack

React 19 + TypeScript + Tailwind CSS 4 + Vite

## Desarrollo

npm run dev

## Compilación

npm run build
